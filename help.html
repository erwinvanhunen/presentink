PresentInk – Help

What is PresentInk?

PresentInk is a macOS menu bar tool for drawing over your screen during presentations or screen sharing.
Use it to highlight, annotate, and focus attention live.

Modes
Command + Shift + D
Toggle active mode
Shift
Draw perfectly straight lines
Command + Shift
Arrow drawing mode
Command
Box drawing mode (rectangle)
Option
Draw ellipses
Cmd + Z
Undo last drawing
Colors
R
Draw in red
G
Draw in green
B
Draw in blue
Y
Draw in yellow
P
Draw in pink
O
Draw in orange
W
Draw in white
Mouse Actions
  • Click and drag to draw (in the active mode)
  • In Arrow/Box mode: click and drag for start/end or corners
  • In Ellipse mode: click and drag for an ellipse
  • Right click: end drawing mode
Status Bar Icon
  • Use the PresentInk icon in your menu bar to open settings, show this help, or quit.
  • When drawing is active, the icon displays background changes to the color selected.
