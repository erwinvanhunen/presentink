What is PresentInk?

PresentInk is a macOS menu bar tool for drawing over your screen during presentations or screen sharing.
Use it to highlight, annotate, and focus attention live.

Keyboard
Option + Shift + D Toggle draw mode on or off
Option + Shift + S Take a rectangular screenshot and copy it to the clipboard.
Shift Draw perfectly straight lines
Cmd + Shift Draw arrows
Cmd Draw boxes (rectangles)
Option Draw ellipses
E Clear all drawings but stay in draw mode
T While in draw mode allows to add text. While type, use cursor up or down to resize the text size
Cmd + Z Undo last drawing
Cmd + Shift + B Break mode. Set the time in the settings
Up Increase line width
Down Decrease line width
Colors
R Draw in red
G Draw in green
B Draw in blue
Y Draw in yellow
P Draw in pink
O Draw in orange
W Draw in white
Mouse Actions
  • Click and drag to draw (in the active mode)
  • In Arrow/Box mode: click and drag for start/end or corners
  • In Ellipse mode: click and drag for an ellipse
  • Right click, ESC or Option + Shift + D: end drawing mode
Status Bar Icon
  • Use the PresentInk icon in your menu bar to open settings, show this help, or quit.
  • When drawing is active, the icon displays background changes to the color selected.
Text Typing

You can use PresentInk to type text for you during presentation. Make sure that you have an application in focus where you can actually type text before you start this functionality. You can use the following keys to type text:

OptionShiftT Start typing text

To use this functionality, you create text file, with a .txt extension. Enter the text as you want it typed. You can break up the text with some keywords. See the example below

            This is the first line of text[end]
            This is the second line of text.[enter]
            And this ends up on a new line.[end]
            [up][up]Now I went two lines up[end]
            Now I wait 3 seconds [pause:3] and then I continue typing[end]

If you execute the example above, text typing will stop at every encounter of the [end] tag. Press the hotkey again to execute the next line.

We support these tags: [up],[down],[left],[right],[enter],[pause:X],[end]
